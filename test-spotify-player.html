🚀 Para Probar con API Real

  1. Crear una aplicación en Spotify Developer Dashboard
  2. Obtener Client ID y Client Secret
  3. Configurar Redirect URI: http://localhost:4321/callback
  4. Agregar las credenciales al archivo .env
  5. Reiniciar el servidor de desarrollo
  6. Hacer clic en "Conectar Spotify" en el reproductor
